# Image Processing (IP)

This repository contains assignments, practicals, and coursework completed as part of the **Image Processing** course during the Second Year of Computer Engineering.

## 📚 Topics Covered

* Digital Image Fundamentals
* Image Representation and Sampling
* Pixel Operations
* Image Enhancement
* Histogram Processing
* Spatial Domain Transformations
* Image Filtering
* Edge Detection
* Image Segmentation
* Morphological Operations
* Feature Extraction
* Color Image Processing
* Applications of Computer Vision

## 📂 Contents

* Laboratory Assignments
* Practical Experiments
* Image Processing Programs
* Mini Projects

## 🛠️ Tools & Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Jupyter Notebook

## 🎯 Learning Outcomes

* Understand the fundamentals of digital image processing.
* Apply image enhancement and filtering techniques.
* Perform edge detection and image segmentation.
* Extract meaningful features from images.
* Develop practical skills using image processing libraries and tools.
