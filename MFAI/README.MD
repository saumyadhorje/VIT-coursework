
# Mathematical Foundations of Artificial Intelligence (MFAI)

This repository contains assignments, practicals, and coursework completed as part of the **Mathematical Foundations of Artificial Intelligence** course during the Second Year of Computer Engineering.

## 📚 Topics Covered

* Propositional Logic
* First Order Predicate Logic (FOPL)
* Logical Reasoning and Inference
* Bayesian Probability
* Conditional Probability
* Bayes' Theorem
* Probability Distributions
* Vectors and Vector Spaces
* Matrices and Matrix Operations
* Linear Algebra
* Eigenvalues and Eigenvectors
* Mathematical Modeling for AI

## 📂 Contents

* Laboratory Assignments
* Practical Exercises
* Problem Sheets
* Mathematical Implementations

## 🛠️ Tools & Languages Used

* Python
* Jupyter Notebook
* NumPy

## 🎯 Learning Outcomes

* Develop the mathematical foundations required for Artificial Intelligence.
* Apply logical reasoning techniques to problem-solving.
* Understand probabilistic models and Bayesian inference.
* Utilize linear algebra concepts in data representation and AI applications.
* Build analytical and mathematical thinking skills for Machine Learning and AI.
