# Data Structures and Algorithms (DSA)

This repository contains assignments, practicals, and implementations completed as part of the **Data Structures and Algorithms** course during the Second Year of Computer Engineering.

## 📚 Topics Covered

* Arrays
* Strings
* Linked Lists
* Stacks
* Queues
* Trees
* Graphs
* Searching Algorithms
* Sorting Algorithms
* Recursion
* Hashing
* Time and Space Complexity Analysis

## 📂 Contents

* Laboratory Assignments
* Practical Programs
* Algorithm Implementations
* Problem-Solving Exercises

## 🛠️ Language Used

* C++

## 🎯 Learning Outcomes

* Understand fundamental data structures and their applications.
* Analyze algorithm efficiency using time and space complexity.
* Implement searching and sorting techniques.
* Develop problem-solving and logical thinking skills.
* Build efficient and optimized solutions for computational problems.
