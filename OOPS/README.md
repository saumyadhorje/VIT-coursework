
# Object Oriented Programming (OOP)

This repository contains assignments, practicals, and programs completed as part of the **Object Oriented Programming** course during the Second Year of Computer Engineering.

## 📚 Topics Covered

* Classes and Objects
* Constructors and Destructors
* Function Overloading
* Operator Overloading
* Inheritance
* Polymorphism
* Virtual Functions
* Friend Functions
* Templates
* Exception Handling
* File Handling

## 📂 Contents

* Assignments
* Practical Programs
* Lab Exercises

## 🛠️ Language Used

* C++ & JAVA

## 🎯 Learning Outcomes

* Understand object-oriented design principles.
* Develop modular and reusable code.
* Apply inheritance and polymorphism effectively.
* Implement advanced C++ programming concepts.
